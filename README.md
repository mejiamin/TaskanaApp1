# TaskanaApp

Приложение для управления задачами с поддержкой сортировки и смены темы.

**Автор:** Магомед-Амин — @mejiamin

**Демо:** [Открыть проект](https://mejiamin.github.io/TaskanaApp1/)

## Технологии

- React
- Vite
- CSS Modules

## Функционал

### Работа с задачами
- Создание задач
- Отображение списка задач
- Редактирование задач
- Удаление задач
- Сохранение и отмена изменений
- Сортировка задач через выпадающее меню

### Удобство и доступность
- Переключение между светлой и тёмной темой
- Навигация по интерфейсу с помощью клавиши Tab
- Добавление задач клавишей Enter
- Отмена действий клавишей Escape
- Всплывающие меню и формы закрываются по клику в пустое место

## 🚀 Быстрый старт

### Клонирование репозитория
```bash
git clone https://github.com/mejiamin/TaskanaApp1.git
```

### Переход в папку проекта
```bash
cd TaskanaApp1
```

### Установка зависимостей
```bash
npm install
```

### Запуск в режиме разработки
```bash
npm run dev
```

### Сборка проекта
```bash
npm run build
```
